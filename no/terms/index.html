Vilkår og Retningslinjer

Denne siden beskriver våre generelle vilkår, avbestillingsregler og personvernerklæring. Ved å bestille en time eller bruke nettstedet vårt, godtar du disse retningslinjene.

Generelle Vilkår

1. Bestillingsregler

  • Alle bestillinger er avhengig av tilgjengelighet.
  • Endring av time må gjøres minst 48 timer i forveien.
  • Avbestillinger må gjøres minst 48 timer i forveien for å unngå avbestillingsgebyr. Se våre avbestillingsregler for fullstendige detaljer.

2. Helse og Sikkerhet

  • Vennligst informer oss om eventuelle allergier, blodrelaterte sykdommer eller hudtilstander før timen din for å ivareta din sikkerhet.

3. Varighet av timen

  • Hver PMU-prosedyre varer vanligvis 60 minutter. Flere prosedyrer kan kreve ekstra tid.

4. Betaling og Gebyrer

  • Full betaling kreves ved fullført prosedyre.
  • Prisene kan endres uten forvarsel.

5. PMU-kontrakt

  • Du må lese og signere vår PMU-kontrakt før timen din. Kontrakten er tilgjengelig på:
    • Engelsk
    • Norsk
    • Russisk

6. Ansvarsfraskrivelse

  • Resultatene av PMU-behandlinger kan variere basert på individuelle hudtyper og etterlevelse av etterbehandlingsinstruksjoner. Vi er ikke ansvarlige for komplikasjoner som følge av ikke-opplyste medisinske tilstander eller manglende overholdelse av etterbehandlingsinstruksjoner.

Avbestillingsregler

Hos annae.tattoo er vi forpliktet til å yte eksepsjonell service til våre kunder. Vi forstår at timeplaner kan endre seg, og våre avbestillingsregler er utformet for å sikre rettferdighet og effektivitet.

1. Avbestillingsfrist

  • Alle bestillinger og avbestillinger må gjøres via vår online bestillingsplattform.
  • Avbestillinger må gjøres minst 48 timer før din planlagte time.
  • Avbestillinger gjort mindre enn 48 timer før timen vil medføre et gebyr på 50 % av den totale bestillingsverdien.

2. Sen Avbestilling og Uteblivelse

  • Sen avbestilling (mindre enn 48 timers varsel) og uteblivelse vil bli belastet med 50 % av den totale bestillingsverdien.
  • Uteblitte timer eller avbestillinger i siste liten hindrer oss i å tilby timen til andre kunder og behandles som sen avbestilling.

3. Endring av Time

  • Timer kan endres opptil 48 timer før timen uten gebyr.
  • Endring innen 48 timer før timen vil ikke medføre gebyr.

4. Refusjoner

  • Forhåndsbetalinger eller depositum vil bli fullt refundert for avbestillinger gjort i henhold til disse reglene.
  • Avbestillinger gjort mindre enn 48 timer før timen kan medføre et fradrag basert på avbestillingsgebyret.

5. Unntak

Vi forstår at nødsituasjoner eller plutselig sykdom kan oppstå. I slike tilfeller, vennligst kontakt oss så snart som mulig for å diskutere alternative løsninger.

6. Kontakt Oss

For spørsmål eller bekymringer om våre avbestillingsregler, vennligst kontakt oss på post@annae.tattoo.

Personvernerklæring

1. Innledning

Hos ALC.XYZ AS (annae.tattoo) er vi forpliktet til å beskytte ditt personvern. Denne personvernerklæringen beskriver hvordan vi samler inn, bruker og beskytter din personlige informasjon når du bestiller en PMU-time gjennom vår plattform.

2. Informasjon Vi Samler Inn

  • Personlig Informasjon: Dette inkluderer ditt navn, kontaktinformasjon (telefonnummer og e-postadresse), og relevant helseinformasjon du oppgir under bestillingsprosessen.
  • Bestillingsinformasjon: Detaljer om timen din, som type prosedyre, dato og klokkeslett.

3. Hvordan Vi Bruker Din Informasjon

  • For å levere tjenester: Vi bruker din personlige informasjon til å administrere timene dine og levere våre PMU-tjenester effektivt.
  • For å kommunisere: Vi kan kontakte deg via e-post eller SMS for å bekrefte timen din, sende påminnelser og gi oppdateringer om bestillingen din.
  • Helse og sikkerhet: Helserelatert informasjon brukes utelukkende for å sikre at din prosedyre er trygg og vellykket.

4. Datasikkerhet

Vi implementerer strenge sikkerhetstiltak for å beskytte din personlige informasjon mot uautorisert tilgang, utlevering, endring eller ødeleggelse.

5. Deling med Tredjeparter

Vi selger, bytter eller deler ikke din personlige informasjon med tredjeparter, unntatt når det er nødvendig for å levere tjenester eller som påkrevd ved lov.

6. Dine Rettigheter

  • Du har rett til å få tilgang til og oppdatere din personlige informasjon.
  • Du kan også be om sletting av dine data med mindre de er nødvendige for juridiske årsaker eller for å fullføre pågående tjenester.

7. Leverandørinformasjon

Vær oppmerksom på følgende tredjepartsleverandører som er involvert i våre tjenester:

  • Betaling i studio: For betalinger i vårt studio bruker vi Zettle, som støtter kortbetaling, Klarna, Vipps og kontanter. Du kan lese Zettles vilkår her.
  • Kalendersynkronisering: Vi synkroniserer avtaler med Google Kalender. Dine avtaledetaljer kan bli lagret eller delt med Google for dette formålet. Du kan lese Googles vilkår her.
  • E-postkommunikasjon: Vi bruker One.com som vår e-postleverandør for å sende bestillingsbekreftelser og varsler. Din e-postadresse og kommunikasjon kan bli behandlet av One.com. Du kan lese deres vilkår her.

8. Endringer i denne erklæringen

Vi kan oppdatere denne personvernerklæringen fra tid til annen. Eventuelle endringer vil bli publisert på nettstedet vårt, og du vil bli varslet via e-post eller SMS.

9. Kontaktinformasjon

For spørsmål eller bekymringer angående vår personvernerklæring, vennligst kontakt oss på privacy@alc.no.
